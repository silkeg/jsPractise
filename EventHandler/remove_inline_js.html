JS Practise, remove inline JS

Task: Remove the complete inline JavaScript which is spreaded all between the markup. The aim is to provide the same functionality by adding just one JS file. To keep it simple, the JS file should be added at the end of the document, to prevent the need for document.ready handler.

In the first step this refactoring should be done without the help of any framework (no jQuery please) - Browser Support: Chrome, Firefox and IE 10

Then, in the next step, please refactor the source again and replace all plain vanilla JS source with the appropriate jQuery functions.
